# NoFuserEx
NoFuserEx is an open-source deobfuscator for ConfuserEx.
License: MIT.

## Usage
Just drag and drop any file(s) onto NoFuserEx.exe.

## Supported protections
* Compressor
* Anti-tamper
* Anti-dumper
* Anti-debugger
* Constants
* Proxy calls
* Working on more
